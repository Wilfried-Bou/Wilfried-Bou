# 👋 Salut, moi c'est Wilfried

## 🚀 À propos de moi

Développeur web diplômé, passionné par la création d'applications et de sites web utiles, performants et maintenables.

Après plusieurs années d'expérience professionnelle dans la restauration et dans le monde des assurances, j'ai choisi de me reconvertir dans le développement web et logiciel où j'ai obtenu les titres professionnels de Développeur Web et Web Mobile (Bac +2) puis de Concepteur Développeur d'Applications (Bac +3/4). Ces formations m'ont permis d'acquérir des compétences solides aussi bien en développement front-end qu'en back-end.

Aujourd'hui, je continue de développer mes compétences à travers des projets personnels, de la veille technologique et la maintenance de sites web pour différents clients.
